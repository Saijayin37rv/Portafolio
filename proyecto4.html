Chatbot de Reclutamiento para Saltus Logistics

¿Cómo se creó?

  1. WhatsApp: Se desarrolló un chatbot usando Python y Node.js para gestionar los webhooks y conectar con la API de OpenAI, permitiendo la interacción y razonamiento de una inteligencia artificial en tiempo real.
  2. Messenger: Se implementó un chatbot similar utilizando JavaScript y Node.js, integrando ngrok para exponer el webhook local y la API de Meta para la comunicación con Messenger.
  3. Asistencia de IA: Todo el desarrollo fue realizado con apoyo de una inteligencia artificial, ya que fue el primer proyecto de este tipo y se requería orientación técnica continua.

¿Cómo funciona?

  • El usuario interactúa con el bot a través de WhatsApp o Messenger.
  • El bot responde utilizando la API de OpenAI, brindando respuestas inteligentes y personalizadas.
  • En WhatsApp, el backend utiliza Python y Node.js; en Messenger, solo JavaScript y Node.js.
  • ngrok se emplea para exponer el webhook local y facilitar la integración con las APIs externas.

Tecnologías utilizadas

  • Python (WhatsApp)
  • Node.js (ambos canales)
  • JavaScript (Messenger)
  • OpenAI API
  • Meta API (Messenger)
  • ngrok
