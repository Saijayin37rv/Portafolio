Descripción general

Esta aplicación móvil, inspirada en plataformas como Didi Food, fue desarrollada especialmente para el local de mi novia y su familia. Permite a los clientes explorar una variedad de productos, realizar pedidos fácilmente y recibir atención personalizada. El sistema está diseñado para ser eficiente, intuitivo y adaptado a las necesidades reales del negocio.

Integración con WhatsApp: Todos los pedidos realizados se envían automáticamente a un número de WhatsApp, facilitando la comunicación directa y el seguimiento entre el cliente y el local.

Características principales

  • Catálogo interactivo de productos con imágenes y descripciones.
  • Flujo de compra sencillo y rápido, con botones funcionales y navegación intuitiva.
  • Historial de pedidos para que el usuario consulte sus compras anteriores.
  • Panel de administración donde se pueden cambiar los estados de los pedidos en tiempo real.
  • Desglose detallado de cada pedido para mejorar la atención y el proceso de entrega.
  • Base de datos y autenticación gestionadas con Firebase.
  • Desarrollada en Flutter usando Android Studio.
  • Preparada para futura publicación en iOS.

Proceso de desarrollo

  1. Aprendizaje intensivo: En dos meses aprendí desde cero sobre variables, funciones, integración de APIs y manejo de bases de datos en Flutter y Firebase.
  2. Desarrollo modular: Implementé cada sección (catálogo, pedidos, historial, admin) asegurando que todos los botones y flujos funcionaran correctamente.
  3. Optimización para el usuario: El diseño y la experiencia están pensados para que cualquier cliente pueda usar la app sin complicaciones.
  4. Escalabilidad: El proyecto está listo para ser publicado en iOS cuando haya presupuesto, y puede crecer según las necesidades del negocio.
Nota personal: Este es el proyecto más grande y significativo que he realizado. Lo desarrollé con mucho empeño y dedicación, ya que es una herramienta real de ayuda para mi novia y su familia, a quienes quiero mucho.
