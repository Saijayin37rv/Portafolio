Menú Virtual - Proyecto Web

Vista principal del menú virtual
Selección de categorías y productos
Creación de pedidos en tiempo real
¿Cómo se creó?
  • Se desarrolló una web app usando HTML, CSS y JavaScript para ofrecer un menú virtual interactivo y responsivo.
  • El menú permite a los clientes seleccionar productos, agregarlos a un carrito y visualizar el total en tiempo real.
  • El proyecto fue el primer paso antes de crear la app móvil, y ayudó a optimizar el tiempo de preparación de los pedidos y mejorar la experiencia del cliente.
  • Se publicó en línea usando GitHub Pages para facilitar el acceso desde cualquier dispositivo.
  • Ver menú en línea

Características principales

  • Interfaz intuitiva y responsiva para cualquier dispositivo.
  • Selección de productos por categorías y visualización de imágenes.
  • Carrito interactivo con actualización automática del total.
  • Facilita la toma de pedidos y reduce errores en la comunicación.

Ejemplo de código del menú interactivo


<div class="menu">
  <div class="menu-item">
    <img src="sushi.jpg" alt="Sushi">
    <h3>Sushi Individual</h3>
    <p>$95</p>
    <button onclick="addToCart('Sushi Individual', 95)">Añadir</button>
  </div>
</div>

<div class="cart">
  <h2>Tu pedido</h2>
  <div id="cart-items"></div>
  <div id="cart-total">Total: $0</div>
</div>

<script>
  const cart = {};
  function addToCart(name, price) {
    if (!cart[name]) cart[name] = { qty: 0, price };
    cart[name].qty += 1;
    renderCart();
  }
  function renderCart() {
    let total = 0;
    const items = Object.entries(cart);
    document.getElementById('cart-items').innerHTML = items.length === 0
      ? '<em>El carrito está vacío.</em>'
      : items.map(([name, item]) => {
          total += item.qty * item.price;
          return `${item.qty} x ${name} <span>$${item.qty * item.price}</span>`;
        }).join('<br>');
    document.getElementById('cart-total').textContent = 'Total: $' + total;
  }
  renderCart();
</script>
