Página Web SEMBRANDO FE

¿Cómo se creó?

  1. Desarrollo con HTML, CSS y JavaScript: La página fue construida desde cero utilizando únicamente tecnologías web básicas, sin frameworks ni gestores de contenido.
  2. Estructura de secciones: Se organizaron diferentes apartados como videos, información sobre la organización, certificaciones y un formulario de contacto.
  3. Ejemplo de buena práctica: Se cuidó la navegación, el diseño responsivo y la claridad visual para ofrecer una experiencia agradable al usuario.
  4. Publicación gratuita: Debido a limitaciones de presupuesto, la página se publicó usando GitHub Pages, un servicio gratuito de hosting para sitios estáticos.

¿Cómo funciona?

  • El usuario puede navegar entre las distintas secciones: inicio, videos, certificaciones y contacto.
  • La sección de videos muestra contenido relevante de la organización.
  • En certificaciones se presentan los logros y reconocimientos obtenidos.
  • El formulario de contacto permite a los visitantes enviar mensajes directamente.

Tecnologías utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • GitHub Pages (hosting gratuito)
